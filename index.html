How does F.I.N.E. work?

There are two main things to know to make the most of your use of this website. The first is that paragraph headers (such as "How does F.I.N.E. work?" above) are links to that paragraph. Not necessarily useful if you're already on the page, but possibly useful when you want to share a specific paragraph with someone. Just right-click the header and copy its link.


The second (and by far the coolest!) thing to know is how to use the code cells on the instruction pages. Every page about C will have at least one code cell like this one:


Copied!

#include <stdio.h>

int main() {
    printf("Hello, welcome to F.I.N.E.!\n");
}

These code cells always contain a full directly executable C program. Right-clicking on a cell copies this correct, immediately compilable script to your clipboard, which means you can paste it with Ctrl + V (if your code editor allows it D:< ). Left-clicking can now still be used to select part of the code in the cell itself. These visible parts in the cells are usually not full programs.
I personally really enjoy how easy these cells are to work with. If you keep an online C editor on hand, you can go from reading to experimenting in no time flat. Give it a shot!
