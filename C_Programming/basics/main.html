The entry point of your program: int main()

Every program in C starts in the main function. This function is always defined with int as the return type (written before the function name), with main as the name of the function and usually with nothing or void inside the parentheses:


Copied!

int main() {
    // Your incredibly amazing code!
}

Note that after the parentheses (), there are braces {} which specify the function body of main. main only executes whatever is written inside of these braces. The following code is incorrect and will cause the compiler to throw an error:


Copied!

#include <stdio.h>

int main() {
    // A (small) masterpiece made by you
}

printf("Hello there!");

Although the command on the last line is correct, the compiler does not accept this program because the command is not included in the braces of a function; in this case main.


Lastly, you are not free to define arguments for main. That is, you cannot ask for any values in the parentheses of main. There are a few exceptions, but those are outside the scope of this basic introduction. All you should know now is that the following is not accepted because it tries to take arguments a and c (in the parentheses right after the function name), which is forbidden:


Copied!

int main(int a, char c) {
    // A (small) masterpiece made by you
}
